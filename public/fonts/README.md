# HarmonyOS Sans SC fonts

This directory contains the Simplified Chinese weights used by the cover
generator. Light, Regular, and Medium are selectable base weights. Bold is
kept for the main-title bold button.

Official design resource:
https://developer.huawei.com/consumer/cn/design/resource/?catalogVersion=V1

Follow Huawei's HarmonyOS Sans distribution terms when redistributing or
modifying these font files.
